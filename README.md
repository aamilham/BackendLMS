# Learning Management System

This is a simple learning management system built with Express.js for the backend and React.js for the frontend.

## Server

Pre-requisites:
- Node.js
- PostgreSQL

```bash
cd server
npm install
npm start
```

## Client

```bash
cd client
npm install
npm start
```
